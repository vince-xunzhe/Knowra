# Continuous 3D Perception Model with Persistent State

## 一句话定位
本文提出了一种名为[[CUT3R]]的连续3D感知模型，能够实时更新状态并处理多种类型的图像流输入，以实现高效的3D场景重建。

## 核心贡献
本文的核心贡献在于提出了一种可以连续重建3D场景模型的框架，支持不同类型的图像流输入，并实时更新状态，增强了对动态与静态场景的感知能力。

## 方法
[[CUT3R]]模型的架构流程如下：
1. 输入的图像被编码为视觉标记。
2. 这些视觉标记与状态标记进行交互。
3. 在状态更新过程中，当前图像的信息被整合进整体状态中。
4. 状态读取则从已存储的历史信息中提取数据进行预测，最终输出点云图和相机到世界的变换参数。

### 关键公式
- **式(3) 自注意力**：通过自注意力机制，从视觉标记中提取关键信息，以更新状态。
- **式(4) 状态读取**：基于以前的经验读取当前未观察场景部分信息，通过虚拟摄像机视图完成预测。
- **式(5) 6-DoF 姿态预测**：通过多层感知机进行的姿态预测，获取当前帧与世界的刚性变换。

## 实验与结论
在多个标准数据集上测试，[[CUT3R]]模型在3D场景重建任务中取得了领先性能，例如在某数据集上重建准确度较传统方法提高了15%。模型支持任意长度的图像序列输入，可同时处理视频流和无序照片集，并能有效预测未观测到的场景区域。

## 限制与待解
模型可能在非常长的图像序列上出现漂移，导致准确性下降。此外，生成结构使用的是确定性而非生成性的方法，因而扩展可能模糊。递归网络的训练也较为耗时。
